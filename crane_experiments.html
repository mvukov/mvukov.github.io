NMPC and MHE to a laboratory scale overhead crane



This was the first application of an auto-generated Nonlinear Model Predictive Control (NMPC) from the CADO toolkit. The experiments are explained in details in this publication. Althought the tracking is slow, we showed that we can run the so-called real-time iteration (RTI) scheme in hard real-time. Furthermore, experiments proved fast convergence rate of the RTI scheme. This implementation used a simple NMPC formulation and a simple estimator.

Results of the improved experiments are published in this paper. We used more detailed model, and more complex formulation for the NMPC. Moreover, this time we used Nonlinear Moving Horizon Estimation (NMHE) to more accurately estimate the system’ states. As a result, faster motions are achieved.
