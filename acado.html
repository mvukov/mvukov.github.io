ACADO toolkit

Toolkit for Automatic Control and Dynamic Optimization | www.acadotoolkit.org

ACADO Toolkit is a software environment and algorithm collection for automatic control and dynamic optimization. It provides a general framework for using a great variety of algorithms for direct optimal control, including model predictive control, state and parameter estimation and robust optimization. ACADO Toolkit is implemented as self-contained C++ code and user-friendly API. The object-oriented design allows for convenient coupling of existing optimization packages and for extending it with user-written optimization routines.

During my PhD days, the developments of ACADO used to be mostly focused around the auto-generated optimization solvers with idea to employ those on fast mechatronics systems. I have spent quite a bit of my time on the solvers for nonlinear MPC and MHE. Compared to the starting point of my developments, at the end of my PhD we achieved the state of far more capable package and an order of magnitude faster generated code. Besides algorithmic implementations, I contributed to the code base maintenance: we moved code repository to Git and set up continuous integration system. And yeah, I migrated the build system from Makefiles to CMake.

More about the code-generation tool (CGT) can be found e.g. in Chapter 4 of my PhD thesis. For other publications on the topic please refer to my Google Scholar profile. Despite being a PhD-grade software package (read: no (unit-)tests, developed mainly for increase of personal publication counters), the tool has been successfully used in a number of academic applications even without any help from the development team – AFAIK. Furthermore, we have learned through various channels that the generated code has been used in some commercial applications. That is brave and certainly contributes to the success of the project.
